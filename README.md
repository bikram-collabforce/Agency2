# Agency2

## React sample app (Railway)

A Railway-ready React sample app lives in **`react-app/`**.

- **Run locally:** `cd react-app && npm install && npm run dev`
- **Deploy to Railway:** See [react-app/README.md](react-app/README.md) for CLI, GitHub, and one-click deploy steps.

The app is a Vite + React starter with a Dockerfile and Caddyfile so Railway can build and serve it as a static site.
